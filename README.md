# Customer Behavior Analytics Dashboard

Python • SQL • Power BI

## Project Overview
This project analyzes customer purchasing behavior to identify revenue drivers, customer segments, subscription trends, and product performance.

Using Python, SQL, and Power BI, I transformed raw customer transaction data into actionable business insights and built an interactive dashboard for data-driven decision making.

## Dataset
The dataset contains 3900 entries of customer transaction and demographic information, including age group, product category, review rating, subscription status and shipping type

## Tools & Technologies

• Python – Data Loading, Cleaning, and EDA
• SQL (PostgreSQL) – Business Analysis
• Power BI – Interactive Dashboard Development

## Project Steps

### 1. Data Loading & Initial Exploration

• Loaded the customer dataset into a Pandas DataFrame
• Examined data types, summary statistics, and missing values

### 2. Data Cleaning & Feature Engineering

• Renamed columns using snake_case formatting for consistency
• Imputed missing review ratings using the median review rating within each product category
• Created an `age_group` feature from the `age` column with the following categories:
  ◦ Young Adult
  ◦ Adult
  ◦ Middle-Aged
  ◦ Senior
• Converted purchase frequency categories into numerical day values
• Removed the redundant `promo_code_used` column

### 3. Database Integration

• Connected Python to PostgreSQL using Psycopg2-binary and SQLAlchemy
• Loaded the cleaned dataset into a PostgreSQL database for querying

### 4. SQL Business Analysis

• Performed business-focused analysis using SQL
• Used aggregations, subqueries, CASE statements, CTEs, and window functions to answer key business questions.

### 5. Dashboard Development

• Built an interactive Power BI dashboard
• Created KPI cards, charts, and filters to visualize customer behavior and revenue insights

## Business Questions

• What is the total revenue generated by each gender?
• Which customers used discounts but still spent above average?
• Which products have the highest average review ratings?
• How do purchase amounts compare between Standard and Express shipping?
• Do subscribed customers spend more than non-subscribers?
• Which products have the highest discount usage rates?
• How are customers segmented into New, Returning, and Loyal groups?
• Are repeat buyers more likely to subscribe?
• What are the top-selling products within each category?
• Which age groups contribute the most revenue?

## Results & Insights

• Clothing generated the highest revenue and sales volume.
• Young Adult and Middle-Aged customers contributed the largest share of revenue.
• Subscription customers showed strong spending behavior.
• Loyal and returning customers represented a significant portion of purchases.
• Several highly rated products consistently performed well across categories.
• Discount usage influenced purchasing behavior without necessarily reducing spending.
